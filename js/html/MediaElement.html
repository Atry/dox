    Direct Subclasses
    AudioElement, VideoElement
    AudioElement

    The HTMLAudioElement interface provides access to the properties of <audio>  elements, as well as methods to manipulate them. It's derived from the HTMLMediaElement interface; it's implemented by nsIDOMHTMLMediaElement .

    VideoElement

    DOM video objects expose the HTMLVideoElement interface, which provides special properties (beyond the regular element object and HTMLMediaElement interfaces they also have available to them by inheritance) for manipulating video objects.

    Documentation for this class was provided by MDN.

    Indirect Subclasses
    Audio
    Audio

    HTML media elements (such as <audio> and <video> ) expose the HTMLMediaElement interface which provides special properties and methods (beyond the regular element object interface they also have available to them by inheritance) for manipulating the layout and presentation of media elements.

    Documentation for this class was provided by MDN.
