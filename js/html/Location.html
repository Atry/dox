    • class Location

    Returns a <a href="#Location_object"> <code>Location</code> object</a>, which contains information about the URL of the document and provides methods for changing that URL. You can also assign to this property to load another URL.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/DOM/window.location">MDN</a>.
