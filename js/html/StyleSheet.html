    CSSStyleSheet
    CSSStyleSheet

    An object implementing the CSSStyleSheet interface represents a single CSS style sheet.

    An object implementing the <code>StyleSheet</code> interface represents a single style sheet.  CSS style sheets will further implement the more specialized <code><a title="en/DOM/CSSStyleSheet" rel="internal" href="/api/js/html/CSSStyleSheet">CSSStyleSheet</a></code> interface.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/DOM/stylesheet">MDN</a>.

    Instance Fields

    var disabled:Bool

    This property indicates whether the current stylesheet has been applied or not.

    var href:String

    Returns the location of the stylesheet.

    var media:MediaList

    Specifies the intended destination medium for style information.

    var ownerNode:Node

    Returns the node that associates this style sheet with the document.

    var parentStyleSheet:StyleSheet

    Returns the stylesheet that is including this one, if any.

    var title:String

    Returns the advisory title of the current style sheet.

    var type:String

    Specifies the style sheet language for this style sheet.
