    • class FEColorMatrixElement

    extends Element

    This filter changes colors based on a transformation matrix. Every pixel's color value (represented by an [R,G,B,A] vector) is <a title="http://en.wikipedia.org/wiki/Matrixmultiplication" class=" external" rel="external" href="http://en.wikipedia.org/wiki/Matrixmultiplication" target="_blank">matrix multiplated</a> to create a new color.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/SVG/Element/feColorMatrix">MDN</a>.
