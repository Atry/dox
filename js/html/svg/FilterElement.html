    • class FilterElement

    extends Element

    The <code>filter</code> element serves as container for atomic filter operations. It is never rendered directly. A filter is referenced by using the <code><a rel="internal" href="https://developer.mozilla.org/en/SVG/Attribute/filter" class="new">filter</a></code> attribute on the target SVG element.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/SVG/Element/filter">MDN</a>.
