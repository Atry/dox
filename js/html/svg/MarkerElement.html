    • class MarkerElement

    extends Element

    The <code>marker</code> element defines the graphics that is to be used for drawing arrowheads or polymarkers on a given <code><a rel="custom" href="https://developer.mozilla.org/en/SVG/Element/path"><path></a></code> , <code><a rel="custom" href="https://developer.mozilla.org/en/SVG/Element/line"><line></a></code> , <code><a rel="custom" href="/api/js/html/svg/PolylineElement"><polyline></a></code> or <code><a rel="custom" href="/api/js/html/svg/PolygonElement"><polygon></a></code> element.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/SVG/Element/marker">MDN</a>.
