    FilterPrimitiveStandardAttributes

    The <code>SVGStylable</code> interface is implemented on all objects corresponding to SVG elements that can have <code><a rel="custom" href="https://developer.mozilla.org/en/SVG/Attribute/style">style</a></code>, {{SVGAttr("class") and presentation attributes specified on them.<br><br> Documentation for this class was provided by <a href="https://developer.mozilla.org/en/DOM/SVGStylable">MDN</a>.
